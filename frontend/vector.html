3. Vector Analyzer for Game/Graphics

🎯 แนวคิด

เครื่องมือวิเคราะห์เวกเตอร์สำหรับงานเกม/กราฟิก

🧠 เนื้อหา
  • เวกเตอร์ 2D/3D
  • ขนาดเวกเตอร์ (Magnitude)
  • Dot / Cross product
  • Unit vector (Normalization)
  • ตรวจขนาน และมุมระหว่างเวกเตอร์
🛠️ Requirement โปรแกรมต้อง:
  • รับเวกเตอร์ 2 ตัว
  • คำนวณ Magnitude, Unit Vector
  • คำนวณ Dot / Cross Product
  • คำนวณมุมระหว่างเวกเตอร์

⭐ Challenge

  • แสดงกราฟเวกเตอร์ (Interactive Canvas)
  • ใช้ใน Simulation การเคลื่อนที่
  • ตรวจว่าเวกเตอร์ตั้งฉากหรือไม่
• Perpendicular IF A · B = 0

💡 Algorithm

• magnitude = sqrt(x² + y² + z²)
• dot = Σ ai * bi
• cross = determinant 3x3
• angle = cos⁻¹(dot / (|A| * |B|))

📥 Input

*3D Isometric View (Projected)

📤 Output

|A| (Magnitude) 0.00
|B| (Magnitude) 0.00
Dot Product 0.00
Cross Product (0, 0, 0)
Angle (Degrees) 0.00°
Perpendicular? NO
Unit Vector A (0, 0, 0)
